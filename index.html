How it works

The Battery Monitor app reads your system's battery using Python's psutil library. It displays a graphical battery bar, percentage, charging status, PC uptime, and app uptime. Every 30 seconds, it logs all information to battery_log.txt for history tracking.

On Linux, it attempts to read battery power in watts from /sys/class/power_supply. On Windows and macOS, wattage may be estimated if available.

Installation & Usage

  1. Download the BatteryMonitor.exe file.
  2. Double-click to run. No installation required.
  3. The GUI will open showing battery status in real time.
  4. Logs will be saved to battery_log.txt in the same folder as the executable.

Notes

  • On Windows, no extra Python installation is needed (bundled in exe).
  • On Linux/macOS, Python 3 and psutil library must be installed if running the .py version.
  • Battery wattage may not be available on all systems.
